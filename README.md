# 🎓 CCMT College Finder

> A smart and student-friendly platform to help candidates during **CCMT Counselling** by providing college insights, cutoff analysis, preference planning, and decision-making tools using publicly available official data.

🌍 **[Visit the Live Website](https://ccmtcollegefinder.web.app)** 

---

## 📌 About the Project

CCMT College Finder simplifies the admission counselling process for students applying through **CCMT (Centralized Counselling for M.Tech / M.Arch / M.Plan)**.

Instead of manually checking PDFs, scattered cutoff files, and multiple pages, students can search, compare, analyze, and plan preferences in one place.

*This project is now open-source to help improve transparency, accessibility, and usefulness for future aspirants.*

## 🤝 Contributing to this Open Source Project

We welcome all developers and students! Open source is a collaborative effort where anyone can inspect, modify, and enhance the project. 

### 💡 What happens in an Open Source Project?
1. **Issues are found or features requested**: Users report bugs or propose ideas.
2. **Discussions happen**: The community decides the best approach.
3. **Code is written**: Developers fork the repository and write the code.
4. **Pull Requests (PRs) are raised**: The code is submitted for review.
5. **Code is merged**: Once approved, your code becomes part of the live project!

### 🛠️ How to Raise a Pull Request

<details>
<summary><b>Click here for step-by-step instructions</b></summary>

1. **Fork the Repository** (Click the "Fork" button at the top right).
2. **Clone your fork locally**:
   ```bash
   git clone https://github.com/YOUR-USERNAME/ccmt-predictor.git
   cd ccmt-predictor
   ```
3. **Create a new branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes** and commit them:
   ```bash
   git commit -m "Add amazing feature"
   ```
5. **Push to your fork**:
   ```bash
   git push origin feature/your-feature-name
   ```
6. **Open a Pull Request** from your fork to our `main` branch.

</details>
